# Hotkey reference

Single source of truth for key bindings. Mirrors `SlopworksInput.inputactions`.
Both devs update this file when adding or changing bindings.

---

## Combat map (FPS gameplay)

Active during normal FPS movement, combat, and exploration.

| Action | Keyboard | Gamepad | Notes |
|--------|----------|---------|-------|
| Move | WASD | Left stick | |
| Look | Mouse | Right stick | |
| Jump | Space | A | |
| Sprint | Shift | L stick click | |
| Interact | E | X | |
| Fire | LMB | RT | |
| Aim | RMB | LT | |
| Reload | R | Y | |
| Inventory | Tab | Start | |
| Switch isometric | V | Select | |
| Toggle build mode | B | -- | Enters Build map |

## Build map (building and factory placement)

Active when build mode is on. Combat map disabled.

| Action | Keyboard | Gamepad | Notes |
|--------|----------|---------|-------|
| Toggle build mode | B | -- | Exits back to Combat map |
| Select tool 1 | 1 | -- | Foundation |
| Select tool 2 | 2 | -- | Wall |
| Select tool 3 | 3 | -- | Ramp |
| Select tool 4 | 4 | -- | Machine |
| Select tool 5 | 5 | -- | Storage |
| Select tool 6 | 6 | -- | Belt |
| Rotate | R | Y | +90 degrees |
| Delete mode | X | -- | Toggle delete highlight |
| Zoop mode | Z | -- | Batch placement toggle |
| Grid overlay | G | -- | Toggle grid lines |
| Cycle variant | Tab | -- | Belt mode or prefab variant |
| Nudge up | PageUp | -- | +1m (+Shift: +0.5m) |
| Nudge down | PageDown | -- | -1m (+Shift: -0.5m) |
| Place | LMB | A | Confirm placement |
| Remove | RMB | -- | Cancel belt / remove |
| Cancel | Esc | B | Cancel pending operation |
| Machine interact | F | X | Cycle recipe on machine |
| Snap filter | Scroll | -- | Toggle center/edge or foundation/peer |
| Debug dump | 0 | -- | Belt diagnostic log |
